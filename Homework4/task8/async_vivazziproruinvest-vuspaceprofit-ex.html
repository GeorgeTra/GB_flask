Полный пример распределения прибыли

Допустим, есть 3 инвестора, каждый из которых в разное время инвестировал:

  • Инвестор_1: $1,000
  • Инвестор_2: 150 000 р.
  • Инвестор_3: 0.5 биткоин

По времени инвестирования: сначала инвестировал Инвестор_1, затем через некоторое время Инвестор_2, а потом Инвестор_3. 

По приглашениям: Инвестор_1 пригласил Инвестора_2. А Инвестор_2 пригласил Инвестора_3. Таким образом, для Инвестора_1 реферал Инвестор_2 будет 1-ого уровня, а реферал Инвестор_3 будет 2-ым уровнем в партнёрской программе.

Допустим, наша компания Vuspace продала продукт или услугу в размере $200 и определила, что с этой продажи будет направлено:

  • 60% ($120) на выплаты по инвестициям "до 150%"
  • 5% ($10) на выплаты по инвестициям "сверх 150%"
  • 20% ($40) по партнёрской программе.
  • остальные 15% в фонд развития компании

Так как осуществилась продажа в долларах, то нужно для расчётов нужно перевести вложения инвесторов в доллары. Пусть на момент инвестирования $1 = 75 рублей, а 1 биткоин = $48,000. Тогда для каждого инвестора его инвестиции в долларах будут:

  • Инвестор_1: $1,000
  • Инвестор_2: $2,000
  • Инвестор_3: $24,000

Распределение на выплаты по инвестициям "до 150%"

Рассмотрим сначала, сколько получат инвесторы, которые суммарно имеют менее 150% в выплатах. По условиям программы те инвесторы, которые инвестировали раньше других, имеют чуть больший процент распределения от наших продаж и соответственно достигают более 150% от инвестиций чуть раньше. В данном примере мы установим следующее правило:

6% от продажи будут распределены так, чтобы те инвесторы, которые инвестировали раньше других получили больше, а остальной процент, то есть 54% будет распределено равномерно.

Равномерное распределение 54%

54% от $200 - это $108, которое будет распределено равномерно. Чтобы узнать какая часть прибыли кому принадлежена, достаточно сумму инвестиции поделить и всю сумму инвестиций и умножить на $108.

Для наглядности представим расчёты в табличной форме:

  Вложенные инвестиции, $ Равномерное распределение, $
Инвестор_1 1000 4   = 108 * 1000 / 27000
Инвестор_2 2000 8   = 108 * 2000 / 27000
Инвестор_3 24000 96   = 108 * 24000 / 27000
Всего: 27000 108

Распределение 6%

Остальные 6%, то есть $12, распределяются так, чтобы инвесторы, которые инвестрировали раньше получили больше части от этих 6%, чем те, которые инвестировали позже.

Для 3 инвесторов распределение будет таким: 3.69%, 2.31% и 0% соответственно:

  Распределение, % Распределение, $
Инвестор_1 3.69 7.38   = 200 * 3.69 / 100
Инвестор_2 2.31 4.62   = 200 * 2.31 / 100
Инвестор_3 0 0   = 200 * 0 / 100
Всего: 6 12

Обратите внимание, что если один инвестор сделал подряд несколько инвестиций, то порядок будет именно по дате первой инвестиции (которые ещё не достигли минимальной ставки в 150%). Таким образом, не будет иметь значение, если инвестор сделает несколько раз подряд инвестиций.

Если вы любите математику и/или очень хотите узнать, как именно были получены эти числа, то щёлкните по ссылке ниже, иначе пропустите расчёты и просто знайте, что часть прибыли которая направлена инвесторам распределяется так, чтобы первые инвесторы получили чуть больше, чем последующие.

Узнать, как распределяется 6% от прибыли инвесторам

Идея заключается в том, чтобы как-то 6% распределить неравномерно, поэтому нужно найти для каждой инвестиции инвесторов такую часть от этого 6%, чтобы другие части других инвесторов были меньше.

Например, для 3-х инвесторов можно было бы распределить так: 4%, 2% и 0% соответственно. А для 5 инвесторов, например так: 4%, 3%, 2%, 1% и 0.

Легче всего для каждого инвестора присваивать некие веса, значение которых зависят от того, когда была сделана инвестиция инвестора по счёту. А потом уже по весам определять, какую часть получит инвестор исходя из своего веса, то есть часть можно найти, как вес делённый на сумму всех весов.

На мой взгляд, интереснее всего 6% распределить по убывающей нелинейной функции, а именно по перевёрнутой параболе, а например, не по линейной. Так бОльшие веса получат самые первые инвесторы.

Формула параболы выглядит следующим образом:

y = a * n^2 + b * n + y0

где:
a < 0, так как парабола перевёрнутая
y0 - некоторое перемещение функции вверх относительно оси OX
y - вес инвестиции инвестора относительно других инвесторов
n - количество инвесторов

Для простоты y0 можно приравнять к 1.

Осталось найти коэффициенты a и b.

Так как значения параболы должны только убывать на положительных значениях n, что в свою очередь говорит о том, что вершина параболы лежит на оси OY, то b равен 0.

Так как последний инвестор имеет вес равный 0, то через приравнивания функции к нулю и x = n можно найти коэффициент a:

0 = a * n^2 + 1
а = - 1 / n^2

Отсюда следуюет, что функция имеет вид:

y = - x^2 / n^2 + 1

По этой формуле распределение 10% для 3-х инвесторов будет:

  Вес Часть от суммы весов Распределение, %
Инвестор_1 0.89   = - 1^2 / 3^2 + 1 0.6154   = 0.89 / (0.89 + 0.56 + 0) 3,69   = 6 * 0.6154
Инвестор_2 0.56   = - 2^2 / 3^2 + 1 0.3846   = 0.56 / (0.89 + 0.56 + 0) 2,31   = 6 * 0.3846
Инвестор_3 0   = - 3^2 / 3^2 + 1 0   = 0 / (0.89 + 0.56 + 0) 0   = 10 * 0
Всего   1 6

Все расчёты верны.

Для 4-х инвесторов будет:

  Вес Часть от суммы весов Распределение, %
Инвестор_1 0.94   = - 1^2 / 4^2 + 1 0.4412   = 0.94 / (0.94 + 0.75 + 0.44 + 0) 2,65   = 6 * 0.4412
Инвестор_2 0.75   = - 2^2 / 4^2 + 1 0.3529   = 0.75 / (0.94 + 0.75 + 0.44 + 0) 2,12   = 6 * 0.3529
Инвестор_3 0.44   = - 3^2 / 4^2 + 1 0.2059   = 0.44 / (0.94 + 0.75 + 0.44 + 0) 1.23   = 6 * 0.2059
Инвестор_4 0   = - 4^2 / 4^2 + 1 0   = 0 / (0.94 + 0.75 + 0.44 + 0) 0   = 10 * 0
Всего   1 6

И так далее.

Итоговое распределение

Представим это в табличной форме:

  Равномерное распределение 54%, $ Распределение 6%, $ Итоговое распределение, $
Инвестор_1 4 7.38 11.38   = 4 + 7.38
Инвестор_2 8 4.62 12.62   = 8 + 4.62
Инвестор_3 96 0 96   = 96 + 0
Всего: 108 12 120

Все значения совпали, следовательно расчёты верны.

На этом этапе также нужно определить достигла ли инвестиция своей ставки инвестирования как минимум 150%:

  Инвестиция, $ Текущие выплаты, $ Ставка минимум 150%, $
Инвестор_1 1000 11.38 1500
Инвестор_2 2000 12.62 3000
Инвестор_3 24000 96 36000

Как видим, в нашем примере на момент первой продажи ни один инвестор не достиг минимальной ставки 150%, следовательно в следующей продаже прибыль с продажи также будут распределяться по их инвестициям. Когда какая-нибудь инвестиция достигнет значения минимальной ставки, то прибыль с продажи по такой инвестиции станет расчитываться по распределению "Свыше 150%". Пример такого распределения представлен ниже.

Распределение на выплаты по инвестициям "свыше 150%"

В нашем примере 5%, то есть $10, идут на выплаты по инвестициям, которые сверх 150%.

Введём в наш пример ещё 2-х инвесторов: Инвестор_А и Инвестор_Б, которые не являются чьми-либо рефералами (чтобы не увеличивать расчёты) и которым компания уже выплатила более 150% от их инвестиций. Пусть Инвестор_А имеет инвестицию в размере $2000, а Инвестор_Б - в размере $5000.

Тогда $10 будут распределяться так:

  Инвестиция, $ Распределение, $
Инвестор_А 2000 2.86   = 10 * 2000 / 7000
Инвестор_Б 5000 7.14   = 10 * 5000 / 7000
Всего: 7000 10

Партнёрское распределение

По условиям партнёрской программы от 10% от продажи направляется на выплату партнёрской программы по уровням 5% - 4% - 3% - 2% - 1% соответственно.

В нашем примере определено 20% от прибыли с продажи, то есть $40, идёт на выплату партнёрской программы, которая будет равномерно распределена между всеми партнёрами.

Сначала представим в табличной форме, сколько каждый инвестор должен получить (так как Инвестор_А и Инвестор_Б не имеют партнёров и не являются рефералами, то они и от них нет партнёрского вознаграждения, поэтому их в расчёт можно не включать):

  Инвестиция, $ Рефералы (уровень относительно пригласителя) Максимум получения партнёрской прибыли, $ Распределение партнёрской прибыли $40, $
Инвестор_1 1,000 Инвестор_2 (1-ый уровень)
Инвестор_3 (2-ый уровень)		 1,060   = 2000 * 0.05 + 24000 * 0.04 18.76   = 40 * 1060 / 2260
Инвестор_2 2,000 Инвестор_3 (1-ый уровень) 1,200   = 24000 * 0.05 21.24   = 40 * 1200 / 2260
Инвестор_3 24,000   0 0   = 40 * 0 / 2260
Всего:     2260 40

Когда будут достигнуто максимальное партнёрское вознаграждение для инвестора, то этот инвестор не участвует в распределении партнёрского вознаграждения, а его излишки будут направленны в выплату по партнёрской программе другим партнёрам.

Выводы денежных средств
